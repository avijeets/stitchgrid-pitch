# StitchGrid Pitch Deck

Standalone investor deck for Vercel.

## Files
- `index.html` — interactive deck
- `StitchGrid-Pre-Seed-Investor-Presentation.pdf` — downloadable PDF for mobile/investors
- `vercel.json` — static hosting + security headers

## Deploy on Vercel
1. Import this repo into Vercel
2. Framework preset: **Other** / static
3. Output directory: leave blank
4. Deploy

## Recommended after deploy
- Enable **Vercel Web Analytics** in the project dashboard
- Add **Deployment Protection** if the plan supports it
- Keep the public URL off indexed surfaces

- The deck includes a `Download PDF` button for mobile
